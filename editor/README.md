# osked
Open Solomon's Key level-editor.

To compile on windows get SFML, Imgui, portable-file-dialogs and generate the following structure on your project folder:

build (folder)
imgui (folder)
----imgui files
include (folder)
----SFML (folder)
--------SFML include files
----portable-file-dialogs.h
lib (folder)
----SFML lib files
res (folder)
func.cpp
globals.cpp
main.cpp
build.bat
